# mrm-frontend
A Medical Records Management system frontend implementation


### Running the project

#### Step 1: Install dependencies
     run npm i
 
#### Step 2: Add environment variables
 
    REACT_APP_BACKEND_BASE_URL=
    
#### Step 3: Run the app

    npm start
    
#### Running in Docker

    docker-compose up -d
